## Instalation
1. Make virtual enviroment
```bash
python -m venv .venv
```
2. Active virtual enviroment
```bash
.\.venv\Scripts\activate 
```
3. Packages
- [Flask](#https://flask.palletsprojects.com/en/3.0.x/)
4. Install packages
``` bash
pip install -r requirements.txt 
```
